## Objective
Learn how Claude can optimize a resume for Applicant Tracking Systems (ATS) and recruiters.

## Key Improvements

- Added a professional summary.
- Improved ATS keywords.
- Rewrote project descriptions with measurable impact.
- Categorized technical skills.
- Removed unnecessary personal information.
- Created a clean one-page ATS-friendly resume.

## Before vs After

| Before | After |
|---------|-------|
| Generic Career Objective | Professional Summary |
| Basic Skills List | Categorized Technical Skills |

| Personal Details Included | ATS-Friendly Format |

---

## What I Learned

- ATS systems prioritize keywords and clean formatting.
- A professional summary improves recruiter understanding.
- Quantified achievements make projects stronger.
- Personal details like DOB and marital status should be removed.
- AI can significantly improve resume quality.
