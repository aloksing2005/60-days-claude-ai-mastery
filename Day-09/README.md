# Day 9 – NutriScope (MVP First Development)

## Objective

Learned how to build an MVP first and then improve it with iterative prompting.

## Stage 1 – MVP

Features:
- Food Input
- Calories
- Protein
- Carbohydrates
- Fat
- Nutrition Score

## Stage 2 – Enhanced

Added:
- BMI Calculator
- Nutrition Charts
- Meal Recommendations
- Water Intake Calculator
- Dashboard Cards
- Export Report
- Premium UI

## MVP vs Enhanced

| MVP | Enhanced |
|------|----------|
| Basic Features | Advanced Dashboard |
| Simple UI | Premium UI |
| Nutrition Analysis | BMI + Charts + Reports |
| Basic Cards | Interactive Dashboard |

## Key Learnings

- Build the MVP first.
- Add features incrementally.
- Use follow-up prompts to enhance applications.
- Claude Artifacts can generate complete HTML apps.
